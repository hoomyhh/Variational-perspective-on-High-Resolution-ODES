Instructions:


run_main.py --optimizer NNAG  --lr 0.05 --a 150

run_main.py --optimizer SGD  --lr 0.01

run_main.py --optimizer SVRG  --lr 0.01



After the training we extract the data for plotting in Latex. The default order of the final data file is SGD , SVRG ,  NNAG , Epoch


Thus, path1 should refer to the path of SGD result 

path2 should refer to the path of SVRG result 

path3 should refer to the path of NNAG result 


And epoch is used for x axis visualisation.


The objective of the plot can be specified as 

“train_error”

“train_acc”

“val_error”

“val_acc”


For example you can run


 load_data.py --path1 “path1” --path2  “path2” --path3 “path3” --obj "val_acc"


Then the data file in the output can be used for visualisation in Latex.
